# TubobubaBot

A Telegram bot that helps download audio from YouTube, Instagram, and TikTok videos.

## Features

- Download MP3 audio from YouTube videos
- Support for Instagram and TikTok (coming soon)
- Automatic file naming with video title
- 64K bitrate for optimized file size

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env` file with your bot token and other configs
4. Run the bot: `npm start`

## Environment Variables

- `BOT_API_KEY`: Your Telegram bot token
- `TG_ID`: Your Telegram user ID (for admin features)
- `BUY_ME_A_COFFEE_URL`: Support link

## Usage

Send a YouTube video link to the bot, and it will process and send the audio file back.
